# Chuleta GIT para Proyectos 🚀

## Comandos básicos

```bash
# Ver estado actual de archivos
git status

# Agregar todos los cambios nuevos o modificados
git add .

# Hacer un commit (guardar cambio en el historial)
git commit -m "Descripción clara del cambio"

# Subir cambios a GitHub
git push

# Traer cambios de GitHub (por si trabajás en varias PCs o en equipo)
git pull

Flujo de trabajo típico
Editá archivos.

git status para ver qué cambió.

git add .

git commit -m "Mensaje"

git push para subir.

Clonar un repositorio en otra PC

git clone https://github.com/tu_usuario/tu_repo.git


Consejos útiles
No subas archivos grandes innecesarios (.mp4, .zip, .iso).

Usá .gitignore para evitar archivos temporales (ejemplo: *.log, build/).

Nunca subas claves privadas, contraseñas ni datos sensibles.
